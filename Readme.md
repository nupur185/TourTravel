# 🌍 **TourTravel **

TourTravel is a simple and user-friendly web application that helps you find and plan the perfect trip based on your preferences. Instead of randomly choosing destinations, this platform guides you with suitable tour options and lets you customize your travel experience easily.

---

## ✨ What this project does

This website is designed to make trip planning easier and more personalized. Whether you already have a destination in mind or you're just exploring ideas, TourTravel helps you discover and book tours that fit your needs.

---

## 🚀 Features
###  Tour Recommendations
- Get suitable travel options instead of randomly choosing places.
###  Tour Guide for Each Destination
- Guide details  
- Pricing  

### 🔍 Filter & Sort Options
- Users can filter and sort tours according to their needs and preferences.

###  Booking System
- You can book tours easily.  
- Bookings are stored using **localStorage** (so they don’t disappear on refresh).

### 📝 Plan Your Trip
- If you're confused, you can use this feature and enter:
  - Your budget  
  - Reason for travel  
  - Preferences
  - and more
- Get better recommendations.

###  Season-Based Categories
- Tours are categorized into:
  - Summer  
  - Winter  
  - Spring  
- Uses Weather API.

###  Responsive Design
- Works properly on mobile, tablet, and desktop screens.

###  Clean UI/UX
- Simple interface, easy navigation, no unnecessary complexity.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Tailwind CSS  
- React.js  
- LocalStorage
